**KINETIC PRINT STUDIOS**
Kinetic is a demo website I built using Node.js, Astro, and Decap CMS, I created a design brief using Gemini, and tried to create the website to it's specification as best as I could. This was an exercise for me to learn how to use Astro and how to use a content management service that wasn't WordPress. Any text or info is taken from the design.md (which is why it might seem AI-ish) but web design is by myself, I went for a brutalist aesthetic
